 # Locust

## Prerequisite

1. Install Python
2. Install the package - `pip3 install locust`
3. Validate your installation - `locust -V`

## Start locust

### 1.Step - Start
```
locust
```
### 2.Step - Web interface
Open http://localhost:8089

### 3.Step - Enter parameter

- Number of users
- Spawn rate 
- Host
